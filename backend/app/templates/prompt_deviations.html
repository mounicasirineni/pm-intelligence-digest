Warning ledger

Warnings fire when the LLM output violates a prompt rule that code cannot deterministically fix. Each warning type has a consecutive-day threshold — reaching it flags a candidate for prompt patching.
Warning What it means Patch threshold
Split Implication 3d A closing sentence states two separate consequences instead of one. The synthesizer is combining implications it should keep separate. 3 consecutive days
Theme Diversity 3d A theme had cross-market items in the WS pool but no paragraph was produced after all fill calls. Coverage gap in What's Shifting. 3 consecutive days
Theme Audit 2d The same theme anchors more than one WS paragraph after deduplication. Dedup should prevent this — indicates upstream routing issue. 2 consecutive days
Routing 2d A source was cited in the wrong section — a dedicated-section source appeared in What's Shifting, or vice versa. 2 consecutive days
